# Install

Set up the project environment: dependencies, env files, database, and verify the app runs.

## Instructions

1. Run `/prime` first — understand the codebase before installing
2. Follow README setup steps exactly
3. Copy `.env.example` → `.env` (and any nested env files) if they don't exist
4. Install all dependencies (backend and frontend) using the project's package manager
5. Run database migrations or seed scripts if documented
6. Start the application in the background and verify it responds
7. Run the test suite once to confirm baseline health
